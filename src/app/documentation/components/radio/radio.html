RadioOption interface

export interface RadioOption {
  label:     string;   // text shown next to the radio button
  value:     any;      // the form control value set when this option is selected
  disabled?: boolean;  // when true, this specific radio is greyed out and non-selectable
}

Individual disabled options do not disable the whole group — the FormControl remains active and other options are still selectable.

Layout comparison

layout="vertical" (default)

Wrapper CSS: flex flex-col gap-2

Best for: 3+ options, long labels, mobile-first forms

layout="horizontal"

Wrapper CSS: flex flex-row flex-wrap gap-4

Best for: 2–4 short options, compact forms (e.g. Yes / No, billing cycles)
