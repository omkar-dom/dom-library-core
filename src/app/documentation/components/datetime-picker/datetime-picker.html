How date and time selections are merged into a single form value.

Merge flow

When date changes (onDateChange)

  1. MatDatepicker emits a Date object
  2. mergeDateAndTime(date, pickerTime()) combines date + HH:mm
  3. setFormattedValue(merged) → formatForValue → control.setValue
  4. pickerDate signal updated for the calendar UI

When time changes (onTimeInput)

  1. Native <input type="time"> emits HH:mm string
  2. pickerTime signal updated
  3. mergeDateAndTime(pickerDate() ?? today, time)
  4. setFormattedValue(merged) → formatForValue → control.setValue

Both paths converge at setFormattedValue(), which is the single write point for the form control.

Stored format examples

format Stored value Notes
"iso" 2025-06-15 14:30 Default. No seconds stored.
"timestamp" 1750000200000 Unix milliseconds. Precise to the minute (seconds zeroed).
"custom" 15/06/2025 14:30 Requires custom_format with both date and time tokens.
