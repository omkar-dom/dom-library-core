How DomMultiSelectComponent handles the "All" toggle and individual selections.

All-toggle decision tree

Previous had __all__? New selection has __all__? Result
No Yes User clicked "All" to select → set [__all__, ...allIds]
Yes No User clicked "All" to deselect → clear to []
Either Either Individual toggle → keep real IDs only, auto-check All if all are now selected

Control value vs on_change payload

control.value (form state)

Array of id_property values, e.g. ["angular", "vue", "__all__"]

__all__ is included when all items are selected — your API layer should filter it out before sending.

on_change payload

Array of the full T objects matching selected IDs (excluding __all__), e.g. [{id:"angular",...}]

Always safe to send directly to your API — contains only real option data.
