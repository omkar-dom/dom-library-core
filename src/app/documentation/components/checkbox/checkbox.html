How data flows through DomCheckboxComponent — including the two render modes.

Render mode decision

options().length === 0

Single checkbox mode

  • • Renders one <input type="checkbox">
  • • Control value is boolean
  • • Use Validators.requiredTrue
  • • Bound directly via [formControl]

options().length > 0

Multi-select checkbox group

  • • Renders N labelled checkboxes
  • • Control value is string[] of IDs
  • • Use Validators.required
  • • State managed via checked_values signal

CheckboxOption interface

export interface CheckboxOption {
  id:    string;   // used as the checkbox value and [id] attribute
  title: string;   // displayed as the option label
}
